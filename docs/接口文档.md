## 开发板直连接口

Arduino已被使用的接口有：

### 供电引脚

该部分涉及的与开发板直接连接的模块，均为通过面包板与开发板连接

| 开发板接口名称 | 模块       | 模块接口 |
|:-------:|:--------:|:----:|
| 5V      | 继电器      | VCC  |
| 5V      | 继电器      | COM  |
| 5V      | OLED显示屏  | VCC  |
| 5V      | 土壤湿度传感器  | /    |
| 5V      | 光照传感器    | /    |
| 5V      | 水位传感器    | /    |
| GND     | 土壤湿度传感器  | /    |
| GND     | 光照传感器    | /    |
| GND     | 水位传感器    | /    |
| GND     | 蜂鸣器模块    | /    |
| GND     | LED灯(电阻) | 负极   |
| GND     | 按钮1      | 右下   |
| GND     | 按钮2      | 右下   |
| GND     | OLED显示屏  | GND  |
| GND     | 水泵       | 负极   |
| GND     | 继电器      | GND  |

### 模拟信号引脚

| 开发板接口名称 | 模块      | 模块接口 |
|:-------:|:-------:|:----:|
| A0      | 光照传感器   | /    |
| A1      | 土壤湿度传感器 | /    |
| A2      | 水位传感器   | /    |
| A4      | OLED显示屏 | SDA  |
| A5      | OLED显示屏 | SCL  |

### 数字信号引脚

| 开发板接口名称 | 模块    | 模块接口 |
|:-------:|:-----:|:----:|
| 2       | 按钮1   | 左上   |
| ~3      | 按钮2   | 左上   |
| 4       | LED灯  | 正极   |
| ~5      | 蜂鸣器模块 | /    |
| ~6      | 继电器   | IN   |

## 继电器引脚

该部分涉及的与开发板直接连接的模块，均为通过面包板与开发板连接

### 控制电路侧

| 接口名称 | 模块  | 模块接口     |
|:----:|:---:|:--------:|
| VCC  | 开发板 | 5V供电     |
| GND  | 开发板 | GND      |
| IN   | 开发板 | ~6数字信号引脚 |

### 被控制电路侧

| 接口名称 | 模块  | 模块接口 |
|:----:|:---:|:----:|
| NC   | 无   | /    |
| COM  | 开发板 | 5V供电 |
| NO   | 水泵  | 正极   |
